# 🍕 Pizza Sales Analytics Dashboard (SQL + Excel)

## 📌 About the Project
This project focuses on analyzing pizza sales data to uncover **business insights, customer behavior, and revenue trends** using **SQL for data processing** and **Excel for visualization**.

The goal is to transform raw transactional data into **actionable insights** for better business decision-making.

---

## 🎯 Problem Statement
Pizza businesses often struggle to:
- Identify top-performing products  
- Understand sales trends across time  
- Track customer ordering patterns  
- Optimize menu and pricing  

This project solves these problems through an **interactive dashboard**.

---

## 🛠️ Tools & Technologies
- **SQL (MySQL)** – Data cleaning, transformation, KPI calculation  
- **Microsoft Excel** – Pivot tables, charts, dashboard, slicers  

---

## ⚙️ Workflow

### 1️⃣ Data Processing in SQL
- Imported raw dataset into MySQL  
- Wrote queries to calculate:
  - Total Revenue  
  - Total Orders  
  - Average Order Value  
  - Total Pizzas Sold  
- Analyzed:
  - Sales by Category & Size  
  - Daily & Hourly Trends  
  - Top & Bottom Sellers  

---

### 2️⃣ Data Export
- Converted SQL outputs into CSV format  
- Imported cleaned data into Excel  

---

### 3️⃣ Dashboard Creation in Excel
- Created Pivot Tables & Charts  
- Built KPI cards:
  - Total Revenue  
  - Total Orders  
  - Avg Order Value  
  - Total Pizzas Sold  
- Added:
  - Timeline (Date filter)  
  - Slicers for interactivity  

---

## 📊 Key Insights
- 📅 Peak sales occur on **weekends (Friday & Saturday)**  
- ⏰ Highest orders between **12 PM – 5 PM**  
- 🍕 **Classic category** generates maximum revenue  
- 📦 **Large size pizzas** sell the most  
- 📉 Some pizzas consistently underperform  

---

## 💡 Business Impact
- Helps optimize staffing during peak hours  
- Identifies best & worst performing products  
- Supports data-driven pricing decisions  
- Improves inventory management
